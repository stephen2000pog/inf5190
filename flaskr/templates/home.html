Projet de session: Programmation Web Avancé

Ce projet réalisé avec Flask consiste à récupérer un ensemble de données provenant de la ville de Montréal et d'offrir des services à partir de ces données. Il s'agit de données ouvertes à propos d'établissements ayant reçu des constats d'infraction lors d'inspections alimentaires.

Fonctionnalités

Un BackgroundScheduler met à jour la base de données tous les soirs à minuit.

Les fonctionnalités implémentées dans la page Rechercher sont :

  • La première page de recherche permet de rechercher les contraventions par: nom d'établissement, propriétaire, rue.
  • La deuxième page de recherche permet de rechercher les contravenants entre deux dates.
  • La troisième page de recherche permet de rechercher les infractions par nom.
